# 자바스크립트에서의 모듈

자바스크립트 모듈을 사용하면 코드를 작은 파일로 나누어서 기능을 export/import할 수 있다.

<br>

- export 선언 : 함수, 객체, 프리미티브 등 모든 것
- import 선언 : 다른 모듈

<br>

> 모듈을 사용하면 <span class="red">재사용성</span>과 <span class="red">유지보수성</span>이 높아진다.

<br>
